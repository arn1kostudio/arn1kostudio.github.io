Chrome Extension Privacy Policy

ArnEQ Chrome Privacy Policy

Effective date: May 11, 2026

Overview

ArnEQ for Chrome is an audio enhancement extension published by Arn1ko Studio. It helps users boost quiet audio, add bass, and improve dialogue clarity on supported websites that play audio or video.

Single purpose

ArnEQ's single purpose is to improve website audio playback by letting users adjust per-site volume, bass, and dialogue boost settings directly in Chrome.

Data stored locally

ArnEQ may store the following information locally in Chrome extension storage:

  • Whether audio processing is enabled or disabled for a site.
  • Saved volume level for a site.
  • Saved bass boost level for a site.
  • Saved dialogue boost level for a site.
  • The website hostname associated with those settings.

This data is stored locally in the user's browser so the extension can restore the user's preferred audio settings when they return to the same site.

Website access

ArnEQ uses host access on standard HTTP and HTTPS pages so it can run its content script on websites selected by the user, detect audio and video elements, and apply audio processing in the active tab.

ArnEQ is not designed to read article text, messages, passwords, form entries, or unrelated page content. It does not use website access to build profiles, advertising segments, or browsing history logs.

Audio processing

Audio processing is performed locally in the browser tab using Chrome and Web Audio features. ArnEQ does not upload audio streams, record microphone input, or send processed audio to Arn1ko Studio.

Website content and browsing history

ArnEQ does not sell user data, does not collect browsing history for analytics or advertising, and does not transmit website content to the developer. Settings are used only to provide the extension's audio enhancement features on a per-site basis.

External links

ArnEQ may open external pages only when the user explicitly requests them, such as the optional donation page at Buy Me a Coffee. Arn1ko Studio does not receive payment data through the extension itself.

Remote code

ArnEQ does not execute remotely hosted code. All JavaScript used by the extension is packaged inside the extension bundle.

Data sharing

ArnEQ does not sell, rent, or share user data with third parties for advertising, analytics, credit scoring, or unrelated purposes. Arn1ko Studio does not receive the locally stored audio settings unless a user separately contacts support and chooses to share details.

Data deletion

Users can reset or change saved site settings inside the extension at any time. Users can also remove the extension from Chrome to delete extension storage associated with ArnEQ.

Chrome Web Store Limited Use

ArnEQ's use of information is intended to comply with the Chrome Web Store User Data Policy, including the Limited Use requirements.

Changes to this policy

This policy may be updated when extension features, permissions, hosting, or Chrome Web Store requirements change. The effective date at the top of this page shows the latest version.
