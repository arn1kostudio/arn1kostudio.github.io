Chrome Extension Privacy Policy

ArnProxy Chrome Privacy Policy

Effective date: May 7, 2026

Overview

ArnProxy for Chrome is a proxy profile manager published by Arn1ko Studio. The extension lets users create, manage, and enable user-provided HTTP/HTTPS and SOCKS5 proxy profiles for Google Chrome. ArnProxy does not provide proxy servers.

Single purpose

ArnProxy's single purpose is to let users create, manage, and enable Chrome proxy profiles. It uses stored profile data only to configure Chrome proxy settings at the user's request, provide proxy authentication for supported HTTP/HTTPS proxies, and show basic before/after IP status in the extension popup.

Data stored locally

ArnProxy may store the following information locally in Chrome extension storage:

  • Proxy profile names.
  • Proxy host, port, and protocol.
  • Proxy username and password, if provided by the user.
  • Active profile selection.
  • Proxy bypass list.
  • Enabled or disabled proxy state.
  • Last before/after IP lookup result shown in the popup.

This data is stored locally in the user's browser. Arn1ko Studio does not receive, sell, share, or use this data for advertising or analytics.

Proxy credentials

Proxy usernames and passwords are used only to authenticate with the proxy server selected by the user. They are stored locally in Chrome extension storage and are not sent to Arn1ko Studio.

Chrome extensions cannot provide username/password authentication for SOCKS5 proxies. SOCKS5 profiles should use IP whitelist access or no username/password authentication.

IP lookup

ArnProxy may contact public IP lookup services such as ipapi.co, ipwho.is, and api.ipify.org to show before/after IP status in the extension popup. These requests are used only for this user-facing feature. The returned IP lookup result may include public IP address, country, city, or network provider details.

Website content and browsing history

ArnProxy does not read website content, does not inject scripts into websites, and does not store browsing history. The extension requests broad host permission so proxy authentication can work for browser requests to any website while a user-selected proxy profile is enabled.

Remote code

ArnProxy does not execute remotely hosted code. All JavaScript code is included inside the extension package. External network requests are used only for public IP lookup services and for opening the optional donation page after the user clicks the Donate button.

Data sharing

ArnProxy does not sell user data and does not share user data with third parties for advertising, analytics, creditworthiness, or any unrelated purpose. Data is used only for the extension's single purpose: managing Chrome proxy profiles.

Data deletion

Users can delete saved proxy profiles inside the extension. Users can also remove the extension from Chrome to delete extension storage associated with ArnProxy.

Chrome Web Store Limited Use

ArnProxy's use of information complies with the Chrome Web Store User Data Policy, including the Limited Use requirements.

Changes to this policy

This policy may be updated when extension features, permissions, hosting, or Chrome Web Store requirements change. The effective date at the top of this page shows the latest version.
