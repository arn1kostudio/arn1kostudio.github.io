Chrome Extension Privacy Policy

TabSpace Chrome Privacy Policy

Effective date: May 13, 2026

Overview

TabSpace for Chrome is a tab session management extension published by Arn1ko Studio. It helps users save the current browser window as a reusable tab space, reopen saved spaces later, organize tabs into groups, and remove duplicate tabs in the current window.

Single purpose

TabSpace has one single purpose: tab session management. All features are limited to saving, restoring, organizing, and cleaning up tab spaces in Chrome.

Data stored locally

TabSpace may store the following information locally in Chrome extension storage:

  • Saved space names.
  • Tab URLs and tab titles included in a saved space.
  • Whether a tab was pinned when the space was saved.
  • The time when a space was created or updated.

This information is stored locally in the user's browser so TabSpace can restore saved spaces and show them in the extension library and launcher.

Tabs and tab groups

TabSpace uses Chrome tab and tab group features to save the current window, reopen saved spaces in a new window, move tabs, group related tabs, and remove duplicate tabs when the user requests those actions.

Browsing activity and website content

To provide its core features, TabSpace reads tab metadata from the user's current window, including tab URLs and tab titles. This is used only to save spaces, restore spaces, detect duplicates, and organize tabs.

TabSpace is not designed to read article text, messages, passwords, form entries, or page content beyond the tab metadata needed for tab management. Arn1ko Studio does not use this information for advertising, analytics, profiling, or unrelated purposes.

Data transmission

TabSpace does not send saved spaces, browsing activity, website content, or tab metadata to Arn1ko Studio servers. The extension does not require an account and does not sync user data to a developer-run backend.

External links

TabSpace may open external pages only when the user explicitly requests them, such as the optional donation page at Buy Me a Coffee. Arn1ko Studio does not receive payment data through the extension itself.

Remote code

TabSpace does not execute remotely hosted code. All JavaScript used by the extension is packaged inside the extension bundle.

Data sharing

TabSpace does not sell, rent, or share user data with third parties for advertising, analytics, credit scoring, or unrelated purposes. Information handled by the extension is used only to provide its tab session management features.

Data deletion

Users can delete saved spaces inside the extension at any time. Users can also remove the extension from Chrome to delete extension storage associated with TabSpace.

Chrome Web Store Limited Use

TabSpace's use of information is intended to comply with the Chrome Web Store User Data Policy, including the Limited Use requirements.

Changes to this policy

This policy may be updated when extension features, permissions, hosting, or Chrome Web Store requirements change. The effective date at the top of this page shows the latest version.
