Proxy utility

ArnProxy

Proxy workflow helper for managing connection profiles and everyday network routing tasks with a focused Android interface.

In development Privacy policy

Purpose

ArnProxy is designed to make proxy profile handling more convenient for users who need a simple mobile tool around proxy workflows.

Expected features

  • Proxy profile management
  • Quick connection state controls
  • Local profile settings
  • Simple status display
  • Designed for practical network workflow use

Privacy

Proxy settings may be stored locally on the device. Network behavior depends on the proxy profile configured by the user.

Status

In development by Arn1ko Studio. Features and release details may change before public distribution.
